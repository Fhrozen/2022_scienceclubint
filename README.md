## Cheatsheet

To execute the notebooks in a personal computer with Windows 10,
install the required programs with the following steps:

    ```batch
    cd installer\win
    install.bat
    ```

Para lanzar los notebooks utilize:

    ```batch
    installer\win\venv\condabin\activate.bat
    jupyter notebook
    ```
